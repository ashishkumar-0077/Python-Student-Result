# 🎓 Student Manager App

A simple and beginner-friendly **Python Command Line Interface (CLI)** application that helps manage student records. Users can add students, view all student records, and check whether a student has passed or failed based on their marks.

This project is ideal for beginners who want to practice Python fundamentals such as dictionaries, loops, conditional statements, and user input handling.

---

## 🚀 Features

- ➕ Add new student records
- 📋 View all student records
- ✅ Check student pass/fail result
- 🔍 Search student by name
- 🚪 Exit the application safely
- 💻 Easy-to-use menu-driven interface
